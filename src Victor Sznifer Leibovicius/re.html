Sensor de Ré


Objetivos :

O objetivo desse projeto foi construir um circuito, com um sensor, amplificadores operacionais, leds, transistores e capacitores, a partir dos conceitos obtidos na disciplina Acionamentos Elétricos.

Utilizamos uma placa Arduino UNO, sensor de ultrassonico, leds, resistores, capacitores, transistores, entre outros. O grupo escolheu fazer uma réplica de sensor de marcha ré de carro, que quando o sensor determinava uma determinada distância, o circuito acendia um led e tocava uma sirene.

Aprendizados:

A parte mais difícil desse projeto foi a conversão de sinais do sensor, pois envolvia muitos passos que precisavam ser muito precisos. Precisamos fazer pelo menos três tentativas para cada sensor. Esse projeto envolveu muita robótica e um pouco de programação para o arduíno, conhecimentos essenciais para qualquer engenharia.
