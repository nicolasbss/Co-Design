Cozimento de um miojo com atividades do dia-a-dia

    Objetivos :

  • O objetivo do projeto era o desenvolvimento de um modelo em python que conseguisse prever quanto tempo demoraria para se cozinhar um miojo a partir de atividades do cotidiano. Para a melhor compreensão aqui vai um exemplo de um dos casos que modelamos: "Quantas vezes é necessário subir uma escada para que a energia gerada seja suficiente para cozinhar um miojo?"

    • Como foi feito:

  • Em python, o modelo foi desenvolvido e testado. Para validar testamos nosso modelo e o comparamos com dados obtidos no mundo real.

    • Aprendizados:

  • Nesse projeto desenvolvemos nossa habilidade de criar modelos para prever comportamentos mundo físico. Aprendi que qualquer modelo seja ele qual for tem limitações.
