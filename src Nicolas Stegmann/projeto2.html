O super Trampolim



    Objetivos:

  • O objetivo do projeto foi a modelagem de um salto de 40 metros em um trampolim. Baseado na angulação do trampolim em relação ao solo, pretendia-se modelar qual a altura que uma pessoa voltaria caso ela pulasse em um trampolim capaz de absorver a queda desta altura.

    • Como foi feito:

  • Para realizar o projeto foi utilizado o python para a programação do modelo físico para a situação.

    • Aprendizados:

  • O projeto, por ser um projeto muito dificil foi uma prova de resiliência. Aprendi que nem tudo ocorre tão facilmente, e é necessário persistência para o sucesso
