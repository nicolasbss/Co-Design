Software de Gerenciamento de Hotéis


Objetivos :

  • O software de gerenciamento de hotéis apresenta uma interface para o hóspede e outra para o funcionário. O hóspede tinha acesso ao serviço de quarto, ao Spa e um local para registrar queixas; além de um portal com as informações do hotel. O funcionário recebia notificações por cada quarto para saber o que cada um desejava e podia checar o que já havia feito. Esse projeto foi muito difícil por termos que aprender a interface TKinter. Além de pensar na lógica tivemos que aplicá-la a esta interface. Foi muito desafiador mas o resultado foi muito gratificante.
